この利用規約（以下「本規約」といいます）は、長野県150周年記念事業実行委員会（以下「当実行委員会」といいます）が運営する長野県150周年記念事業サイト（以下「本サイト」といいます）の利用に関する条件を定めるものです。本サイトをご利用の際は、本規約をお読みいただき、同意の上でご利用ください。

第1条（総則）

  1. 本規約は、本サイトの利用に関して、当実行委員会と利用者との間の権利義務関係を定めることを目的とします。
  2. 本サイトをご利用いただくことにより、利用者は本規約に同意したものとみなされます。
  3. 本規約に同意いただけない場合は、本サイトのご利用をお控えください。
  4. 本規約は、本サイトのすべてのサービスおよび機能に適用されます。

第2条（定義）

本規約において、次の各号に掲げる用語の意義は、当該各号に定めるところによります。

1. 当実行委員会

長野県150周年記念事業実行委員会をいいます。

2. 本サイト

当実行委員会が運営する長野県150周年記念事業サイト（https://nagano-150th-a.vercel.app）およびそのサブドメインを含むすべてのウェブサイトをいいます。

3. 利用者

本サイトにアクセスし、または本サイトを利用するすべての個人および法人をいいます。

4. コンテンツ

本サイトに掲載される文章、画像、動画、音声、ソフトウェア、その他の情報をいいます。

5. 個人情報

個人情報保護法に定める個人情報をいいます。

6. サービス

本サイトを通じて当実行委員会が提供するすべての機能およびサービスをいいます。

第3条（規約の同意）

  1. 利用者は、本サイトを利用することにより、本規約に同意したものとみなされます。
  2. 未成年者が本サイトを利用する場合、法定代理人（親権者等）の同意が必要です。
  3. 法人が本サイトを利用する場合、その代表者または担当者が法人を代表して本規約に同意する権限を有していることを保証するものとします。
  4. 本規約に同意いただけない場合は、本サイトのご利用を直ちに中止してください。

第4条（サービス内容）

本サイトでは、以下のサービスを提供しています。

📚 情報提供サービス

  • 長野県150周年記念事業に関する情報提供
  • 長野県の歴史に関する資料・情報の提供
  • イベント・催事の案内・情報提供
  • 特集記事・コンテンツの提供

📝 申込・お問い合わせサービス

  • イベント参加申込の受付
  • お問い合わせフォームの提供
  • メール配信サービスの提供
  • 各種申請・届出の受付

🔍 検索・閲覧サービス

  • サイト内検索機能の提供
  • コンテンツアーカイブの提供
  • サイトマップの提供
  • 関連リンクの提供

📱 その他のサービス

  • モバイル対応サービス
  • アクセシビリティ対応
  • 多言語対応（予定）
  • その他当実行委員会が必要と認めるサービス

第5条（利用規則）

利用者は、本サイトを利用するにあたり、以下の事項を遵守するものとします。

✅ 基本的な利用マナー

  • 法令、本規約およびマナーを遵守して利用すること
  • 他の利用者や第三者の権利を侵害しないこと
  • 虚偽の情報を提供しないこと
  • 公序良俗に反する行為を行わないこと

🔒 セキュリティ関連

  • 本サイトのセキュリティを脅かす行為を行わないこと
  • 不正アクセスやシステムの妨害行為を行わないこと
  • ウイルスやマルウェアの配布を行わないこと
  • パスワード等の認証情報を適切に管理すること

📄 コンテンツ利用関連

  • 著作権、商標権等の知的財産権を侵害しないこと
  • コンテンツの無断転載・複製を行わないこと
  • 商用利用を行う場合は事前に許可を得ること
  • 出典を明記せずに引用しないこと

💬 コミュニケーション関連

  • 誹謗中傷、差別的な発言を行わないこと
  • スパム行為を行わないこと
  • 営利目的の宣伝・勧誘を行わないこと
  • 政治的・宗教的な主張を過度に行わないこと

第6条（禁止行為）

利用者は、本サイトの利用にあたり、以下の行為を行ってはなりません。

🚫 法令違反・権利侵害

  • 法令または公序良俗に違反する行為
  • 犯罪行為に関連する行為
  • 他者の著作権、肖像権、プライバシー権等を侵害する行為
  • 他者の名誉、信用を毀損する行為
  • 差別や偏見を助長する行為

💻 システム妨害

  • 本サイトの運営を妨害する行為
  • サーバーやネットワークに負荷をかける行為
  • 不正アクセス、ハッキング等の行為
  • ウイルス、マルウェア等の配布
  • セキュリティホールを悪用する行為

📄 不適切なコンテンツ

  • 虚偽、デマ、誤解を招く情報の提供
  • わいせつ、暴力的、反社会的なコンテンツの投稿
  • スパム、迷惑行為
  • 無関係な宣伝・広告の投稿
  • チェーンメール等の投稿

🎭 なりすまし・詐欺

  • 他者になりすます行為
  • 虚偽の身元を申告する行為
  • 詐欺的な行為
  • マルチ商法等の勧誘行為
  • 個人情報を不正に取得しようとする行為

⚖️ その他

  • 本サイトの趣旨に反する行為
  • 反社会的勢力との関わりを持つ行為
  • 未成年者に有害な情報の提供
  • 当実行委員会の業務を妨害する行為
  • その他、当実行委員会が不適切と判断する行為

第7条（コンテンツの取り扱い）

📝 当実行委員会のコンテンツ

  1. 本サイトに掲載されるコンテンツの著作権は、当実行委員会または正当な権利者に帰属します。
  2. 利用者は、私的使用の範囲内で本サイトのコンテンツを利用することができます。
  3. 商用利用、二次利用を行う場合は、事前に当実行委員会の許可を得る必要があります。
  4. コンテンツを引用する場合は、適切な出典表記を行うものとします。

👤 利用者提供コンテンツ

  1. 利用者が本サイトに投稿・送信したコンテンツについて、当実行委員会は以下の権利を行使できるものとします：
    • 本サイトでの掲載・表示
    • 事業目的での利用
    • 必要に応じた編集・改変
    • 他のメディアでの二次利用
  2. 利用者は、投稿するコンテンツについて必要な権利を有していることを保証します。
  3. 当実行委員会は、投稿されたコンテンツの内容について責任を負いません。
  4. 不適切と判断されるコンテンツは、予告なく削除することがあります。

🔗 第三者コンテンツ

  1. 本サイトには第三者のウェブサイトへのリンクが含まれる場合があります。
  2. リンク先の内容について、当実行委員会は責任を負いません。
  3. 利用者は、リンク先サイトの利用規約に従って利用するものとします。
  4. 外部サイトとの間で生じたトラブルについて、当実行委員会は関与いたしません。

第8条（個人情報）

  1. 当実行委員会は、利用者の個人情報をプライバシーポリシーに従って適切に取り扱います。
  2. 利用者は、個人情報の提供にあたり、プライバシーポリシーの内容を確認し、同意するものとします。
  3. 当実行委員会は、個人情報保護法その他関連法令を遵守して個人情報を取り扱います。
  4. 利用者の個人情報に関する問い合わせは、プライバシーポリシーに記載の方法で受け付けます。

第9条（責任の制限）

🛡️ 当実行委員会の責任範囲

  1. 当実行委員会は、本サイトの利用により利用者に生じた損害について、故意または重大な過失がある場合を除き、責任を負いません。
  2. 本サイトの情報の正確性、完全性、有用性について保証するものではありません。
  3. サービスの中断、停止により生じた損害について責任を負いません。
  4. 利用者間または利用者と第三者との間のトラブルについて責任を負いません。

👤 利用者の責任

  1. 利用者は、本サイトの利用により第三者に損害を与えた場合、自己の責任において解決するものとします。
  2. 利用者の行為により当実行委員会が損害を被った場合、利用者がその損害を賠償するものとします。
  3. 利用者は、パスワード等の管理について責任を負うものとします。
  4. 未成年者の利用については、法定代理人が責任を負うものとします。

第10条（免責事項）

当実行委員会は、以下の事項について責任を負いません。

🌐 システム・技術関連

  • サーバーの停止、メンテナンス等によるサービス中断
  • 通信回線の障害によるアクセス不能
  • コンピュータウイルス等による被害
  • 技術的な不具合による情報の消失

📄 情報・コンテンツ関連

  • 掲載情報の正確性、完全性、有用性
  • 情報の更新遅れや掲載漏れ
  • 外部リンク先の内容や安全性
  • 利用者が投稿したコンテンツの内容

👥 利用者関連

  • 利用者同士のトラブル
  • 利用者と第三者との間のトラブル
  • 利用者の本サイト利用による経済的損失
  • 機会損失、精神的苦痛等の間接的損害

⚡ 不可抗力

  • 天災、戦争、暴動等の不可抗力による影響
  • 法令の改正による影響
  • 行政機関の処分による影響
  • その他当実行委員会の責に帰すべからざる事由

第11条（サービスの変更・停止）

🔄 サービスの変更

  1. 当実行委員会は、事前の通知なくサービス内容を変更することができます。
  2. 重要な変更については、可能な限り事前に通知するよう努めます。
  3. 変更により利用者に不利益が生じても、当実行委員会は責任を負いません。

⏸️ サービスの一時停止

以下の場合、事前の通知なくサービスを一時停止することがあります：

  • システムメンテナンスのため
  • システム障害が発生したため
  • 通信回線等の障害のため
  • 自然災害等の不可抗力のため
  • その他運営上必要と判断したため

🛑 サービスの終了

  1. 当実行委員会は、本サイトのサービスを終了することがあります。
  2. サービス終了の場合は、可能な限り事前に通知いたします。
  3. サービス終了により利用者に損害が生じても、当実行委員会は責任を負いません。
  4. 長野県150周年記念事業の終了に伴い、本サイトも終了する予定です。

第12条（規約の変更）

  1. 当実行委員会は、必要に応じて本規約を変更することがあります。
  2. 規約の変更は、本サイトでの公表により効力を生じます。
  3. 重要な変更については、可能な限り事前に通知いたします。
  4. 変更後の規約は、公表された時点で効力を生じ、それ以降の本サイト利用に適用されます。
  5. 変更された規約に同意いただけない場合は、本サイトのご利用をお控えください。

第13条（準拠法・管轄裁判所）

  1. 本規約の準拠法は日本法とします。
  2. 本規約または本サイトの利用に関して紛争が生じた場合、当実行委員会の所在地を管轄する裁判所を専属的合意管轄裁判所とします。
  3. 紛争の解決にあたっては、まず誠意をもって協議による解決を図るものとします。
  4. 協議による解決が困難な場合は、上記管轄裁判所にて解決を図るものとします。
