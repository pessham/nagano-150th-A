長野県150周年記念特集記事シリーズ 第3回

結いの精神が育む包容力のあるコミュニティ― 誰もが活躍できる長野県の実現 ―

🤝 コミュニティと互助
読了時間：約18分 2025年10月公開

「結い」とは、農村社会で育まれた相互扶助の精神です。現代の長野県では、この伝統的な支え合いの文化が、多文化共生や移住者受け入れ、高齢者活躍支援など、新しい形のコミュニティづくりに活かされています。誰もが居場所と出番を見つけられる包容力のある社会は、どのように実現されているのでしょうか。

結いの精神と現代的な互助システム

長野県の農村地域で古くから根付く「結い」は、田植えや稲刈り、屋根の葺き替えなど、一人では困難な作業を地域全体で助け合う相互扶助システムです。この精神は現代においても形を変えながら受け継がれ、新しいコミュニティづくりの基盤となっています。

結いから生まれた現代的な支援システム

佐久市では、伝統的な結いの精神を活かした「地域支え合い協議会」が全26地区で設立されています。高齢者の見守り活動から子育て支援、災害時の助け合いまで、住民主体の多様な取り組みが展開されています。

佐久市・望月地区の取り組み

  • 安心カード配布: 一人暮らし高齢者の緊急連絡先や持病を記録
  • お互いさまの会: 住民同士の日常的な支援マッチング
  • 防災ネットワーク: 災害時の避難支援体制を地域全体で構築

この取り組みの特徴は、支援を受ける側も必ず何らかの形で地域に貢献する「お互いさま」の関係を重視していることです。一方的な支援ではなく、それぞれが持つ技能や知識を活かし合う双方向の関係が、持続可能なコミュニティを作り上げています。

多文化共生の最前線

長野県には約58,000人の外国人住民が暮らしており（2024年現在）、全人口の約2.8%を占めています。製造業を中心とした就労や国際結婚により定住する外国人が増加する中、地域社会との共生に向けた取り組みが各地で進んでいます。

飯田市の多文化共生推進

飯田市では、ブラジル系住民を中心とした外国人コミュニティとの共生に20年以上取り組んできた実績があります。2019年に策定された「飯田市多文化共生社会推進基本方針」に基づき、言語の壁を越えた地域参加の仕組みづくりが進んでいます。

長野県内の外国人住民数の推移（主要国籍別）

2024年データ:

  • 中国: 15,200人（26.2%）
  • ブラジル: 10,800人（18.6%）
  • ベトナム: 8,900人（15.4%）
  • フィリピン: 6,100人（10.5%）
  • その他: 17,000人（29.3%）

言語を超えたコミュニケーション

上田市の「うえだ多文化共生推進協会」では、日本語教室の運営だけでなく、料理交流会や文化祭への参加など、言語の壁を越えた交流の場を創出しています。特に注目されるのは、外国人住民が講師となって母国の文化を紹介する「逆向き」の文化交流です。

成功事例：上田市の国際文化交流祭

毎年秋に開催される国際文化交流祭では、20カ国以上の外国人住民が母国料理の出店や伝統芸能の披露を行います。来場者数は年々増加し、2023年には約15,000人が参加。日本人住民にとっても多様な文化に触れる貴重な機会となっています。

移住者を迎えるコミュニティ力

長野県への移住者数は年々増加しており、2023年度の移住者数は過去最高の2,896人を記録しました。この背景には、移住者を温かく迎え入れる地域コミュニティの力があります。

軽井沢町の移住支援システム

軽井沢町では、移住前の相談から定住後のフォローアップまで、段階的な支援システムを構築しています。特に注目されるのは「移住サポーター制度」で、既に移住した住民が新たな移住者のメンターとなる仕組みです。

軽井沢町の移住支援プロセス

  1. 移住相談会: 東京・大阪で年6回開催、現地見学ツアーも実施
  2. お試し移住: 最大3ヶ月間、町営住宅を格安で提供
  3. 移住サポーター: 先輩移住者が生活情報や人脈を提供
  4. コミュニティ参加: 町内会や趣味クラブへの参加をサポート
  5. 定期フォロー: 移住後3年間の定期的な相談対応

小布施町の「関係人口」創出

小布施町では、移住に至らなくても町と継続的な関わりを持つ「関係人口」の創出に力を入れています。「小布施若者会議」や「小布施見にマラソン」など、町の魅力発信と関係性構築を両立させたイベントが人気を集めています。

「移住は人生の大きな決断です。私たちができるのは、その決断を支え、新しい環境で安心して暮らしていけるよう、地域全体で見守ることです。一人ひとりが地域の大切な仲間として迎え入れられる、そんな温かいコミュニティでありたいと思っています。」
— 軽井沢町移住サポーター 田中美保さん（2018年東京から移住）

高齢者が活躍する地域づくり

長野県は全国でも有数の長寿県として知られていますが、単に長生きするだけでなく、高齢者が地域社会で活躍し続けられる環境づくりに取り組んでいます。高齢化率31.5%（2023年）の現実を「経験と知恵の宝庫」として前向きに捉える取り組みが各地で展開されています。

松本市の「生涯現役促進プロジェクト」

松本市では、60歳以上の市民を対象とした「セカンドライフ支援センター」を設置し、退職後の多様な生き方を支援しています。就労支援、ボランティア活動の紹介、起業支援など、一人ひとりの希望に応じたセカンドキャリアの選択肢を提供しています。

松本市セカンドライフ支援センター実績（2023年度）

  • 相談件数: 1,247件（前年比15%増）
  • 就労マッチング: 312件成立
  • ボランティア登録: 468人
  • 起業支援: 23件（内、実際の起業12件）

伊那市の「高齢者技能継承プロジェクト」

伊那市では、熟練技能者や職人の知識・技術を次世代に継承する取り組みが活発です。「ものづくりマイスター制度」では、優れた技能を持つ高齢者が地域の若者や移住者に技術指導を行っています。

事例：竹細工職人・山田晴夫さん（78歳）

50年以上竹細工に携わる山田さんは、月2回の竹細工教室を主宰。これまでに60人以上の生徒を指導し、そのうち8人が本格的に竹細工を始めました。「技術を教えることで、逆に若い人から新しいアイデアをもらえる。お互いに学び合える関係が一番いいですね」と語ります。

デジタル化時代のコミュニティ形成

新型コロナウイルス感染症の影響により、地域コミュニティのあり方も大きく変化しました。長野県では、デジタル技術を活用した新しい形のコミュニティ形成が進んでいます。

塩尻市の「デジタル田園都市構想」

塩尻市は「デジタル田園都市構想」の先進自治体として、ICTを活用した地域課題解決に取り組んでいます。市民同士をつなぐプラットフォーム「しおじりコネクト」では、困りごとの相談や特技の提供、イベント情報の共有などがオンラインで行われています。

「しおじりコネクト」の主な機能

  • 困りごと相談: 日常的な悩みを住民同士で解決
  • スキルシェア: 得意分野を活かした住民間の助け合い
  • イベント共有: 地域の催し物や活動の情報発信
  • 防災情報: 緊急時の安否確認と情報共有
  • 子育て支援: 育児情報の交換と相互支援

オンラインとオフラインの融合

重要なのは、デジタル技術によってリアルなコミュニティが置き換わるのではなく、両者が補完し合う関係を築くことです。茅野市では、オンラインでの情報共有やマッチングを経て、実際の対面活動につなげる「ハイブリッド型コミュニティ」の形成を支援しています。

茅野市「ちのコミュ」の取り組み

地域SNS「ちのコミュ」で知り合った住民が、実際に地域の清掃活動や防災訓練、お祭りの準備などで協力し合う事例が増加。オンラインでの「顔見知り」関係が、リアルな地域活動の活性化につながっています。登録者数は開設2年で3,200人を突破し、月間アクティブユーザーは1,800人に達しています。

NPO・市民活動の活性化

長野県は人口当たりのNPO法人数が全国で2位（2024年現在、1,089法人）と、市民活動が非常に活発な地域です。この背景には、住民主体で地域課題に取り組む文化的土壌があります。

長野市「市民活動支援センター」の役割

長野市では、NPOや市民団体の活動を総合的に支援する「市民活動支援センター」を運営しています。団体設立の相談から活動場所の提供、資金調達の支援まで、幅広いサポートを行っています。

市民活動支援センターの支援メニュー

  • スタートアップ支援: 団体設立から活動開始までのサポート
  • スペース提供: 会議室や作業スペースの格安貸出
  • 資金調達支援: 助成金申請のサポートと独自の助成制度
  • ネットワーキング: 団体間の連携促進イベント
  • 人材マッチング: ボランティアと団体のマッチング

協働によるイノベーション

特筆すべきは、行政・企業・市民団体が連携した「協働」による課題解決が進んでいることです。大町市では、移住促進・空き家対策・観光振興を一体的に進める「おおまち暮らし向上委員会」が、多様な主体の連携により成果を上げています。

大町市「おおまち暮らし向上委員会」の成果

  • 空き家活用: 3年間で45軒を移住者向け住宅として再生
  • 移住促進: 年間移住者数が3年で2倍に増加（78人→156人）
  • 観光振興: 移住者の視点を活かした新たな観光コンテンツ開発
  • 雇用創出: 移住者による新規事業立ち上げ12件

参加者の声

「一人では解決できない地域の課題も、みんなで知恵を出し合えば必ず道は見つかります。行政の制度、企業の技術、市民の熱意、それぞれの強みを活かした協働こそが、持続可能な地域づくりの鍵だと実感しています。」
— おおまち暮らし向上委員会代表 佐藤健太さん

まとめ：共生と活躍の未来

長野県における包容力のあるコミュニティづくりの根底には、「結い」に象徴される相互扶助の精神が流れています。この伝統的な価値観を現代的にアップデートすることで、多文化共生、移住者支援、高齢者活躍、デジタル活用、市民協働など、多様な分野で先進的な取り組みが生まれています。

長野県モデルの特徴

1. 双方向性

一方的な支援ではなく、誰もが支える側にも支えられる側にもなる「お互いさま」の関係

2. 多様性の受容

国籍、年齢、性別、障がいの有無に関わらず、一人ひとりの個性と能力を活かす場の創出

3. 協働によるイノベーション

行政・企業・市民団体・住民が連携し、それぞれの強みを活かした課題解決

4. 伝統と革新の融合

古くからの知恵と最新技術を組み合わせた持続可能なコミュニティづくり

未来への展望

人口減少と高齢化が進む中、長野県のコミュニティづくりは全国のモデルケースとして注目されています。重要なのは、制度や仕組みを整えるだけでなく、一人ひとりが地域の一員として当事者意識を持ち、積極的に関わることです。

「誰一人取り残さない」長野県の実現に向けて、今後も多様な主体が連携し、創意工夫を重ねながら、より包容力のあるコミュニティを築いていくことが期待されます。そこには、新しい時代の「結い」の形があるはずです。

2030年に向けた目標

  • 外国人住民の地域参加率: 現在の35% → 60%に向上
  • 移住者の定住率: 現在の78% → 90%に向上
  • 高齢者の社会参加率: 現在の45% → 70%に向上
  • デジタル活用率: 全世代の80%がオンラインコミュニティに参加
